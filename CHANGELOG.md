## [2.0.0] - 2025-01-11

#### 核心功能
- **Token管理模块**：实现安全的Token存储和验证

  - Token格式验证
  - 连接测试功能
  - 支持Token更新和删除

- **参数优化器**：全新的参数优化系统
  - 支持暴力搜索算法
  - 支持遗传算法
  - 支持粒子群算法
  - 蓝图管理系统
  - 结果可视化分析

- **权重配置系统**：增强的权重管理
  - 主权重配置（7个核心指标）
  - 子权重配置（高级功能）
  - 配置保存、加载、删除、复制
  - JSON格式存储

### 迁移指南 🔄

#### 从v1.0.0升级到v2.0.0

1. 备份现有配置和数据
2. 更新代码到v2.0.0
3. 安装新的依赖：`pip install -r requirements.txt`
4. 启动应用，系统会提示迁移Token
5. 在Token配置界面点击「迁移旧Token」
6. 验证功能正常后，删除旧的`token_config.py`

#### Token迁移

如果使用过旧版本的`token_config.py`：

```bash
# 启动应用
python launcher.py

# 在Web界面的Token配置区域
# 点击「从旧系统迁移」
# 点击「迁移旧Token」
```
