## 策略控制器使用

策略控制器是核心选股系统，提供策略配置、选股执行、结果查看和回测分析功能。

### 1. 启动策略控制器

#### 方法A：使用启动器（推荐）

```bash
python launcher.py
```

选择「1. 启动策略控制器」或「3. 同时启动两个应用」。

#### 方法B：直接启动

```bash
streamlit run strategy_controller/main.py --server.port 8502
```

### 3. Token配置

首次使用必须配置API Token：

#### 步骤

1. 在左侧边栏找到「🔑 API Token配置」
2. 点击「配置Token」展开配置面板
3. 粘贴从东财掘金终端获取的Token
4. （可选）添加Token描述，便于识别
5. 点击「💾 保存Token」
6. 系统会自动验证Token格式

#### Token验证

- **格式验证**：检查Token长度和字符格式
- **连接测试**：尝试使用Token连接东财掘金API

### 4. 策略选择

当前支持的策略：

- **z哥综合策略 (KDJ+知行趋势+深V信号)**
  - KDJ指标：超买超卖信号
  - 知行趋势：趋势跟踪信号
  - 深V信号：反转信号

### 5. 权重配置

#### 主权重配置

调整7个核心指标的权重：

| 指标 | 默认权重 | 说明 |
|------|---------|------|
| kdj_j | 25 | KDJ的J值，反应超买超卖 |
| trend | 25 | 知行趋势指标 |
| deepv | 10 | 深V反转信号 |
| volume | 8 | 成交量指标 |
| fundamental | 8 | 基本面指标 |
| position | 4 | 持仓比例 |
| risk_reward | 20 | 风险收益比 |

**注意**：所有权重之和应为100。

#### 子权重配置（高级功能）

某些指标可以进一步细分子权重，例如：

- **trend**：可以细分为短期趋势、中期趋势、长期趋势
- **volume**：可以细分为成交量放大、成交额等

启用方法：
1. 在权重配置区域点击「配置子权重」
2. 为每个主指标设置子权重
3. 系统会自动计算最终评分

#### 配置管理

**保存配置**：
1. 调整权重后，点击「💾 保存配置」
2. 输入配置名称和描述
3. 配置会保存到`web/configs/weight_configs.json`

**加载配置**：
1. 点击「📂 加载配置」
2. 从列表中选择已保存的配置
3. 权重会自动加载到界面

**删除配置**：
1. 点击「🗑️ 删除配置」
2. 选择要删除的配置
3. 确认删除

**复制配置**：
1. 点击「📋 复制配置」
2. 选择要复制的配置
3. 输入新配置名称
4. 修改权重后保存

### 6. 筛选参数

在「筛选参数」区域设置：

#### 基本参数

- **最大结果数**：返回的最大股票数量（默认：30）
- **跳过ST股**：是否排除ST股票（默认：勾选）
- **测试模式**：快速测试，仅处理少量股票（默认：不勾选）

#### 性能参数

- **批量大小**：每次批量处理的股票数（默认：1000）
- **并发数**：并发处理的线程数（默认：6）
- **股票池类型**：选择股票池范围
  - 全量A股
  - 沪深300
  - 中证500
  - 中证1000

### 7. 执行选股

#### 开始选股

1. 配置好所有参数
2. 点击「🚀 开始选股」按钮
3. 系统会显示实时进度和状态信息
4. 等待选股完成

#### 进度显示

- 进度条显示选股完成百分比
- 状态文本显示当前处理阶段
- 支持长时间运行的选股任务

#### 选股结果

选股完成后，主界面会显示：

1. **结果摘要**：
   - 上次筛选时间
   - 找到的股票数量

2. **结果表格**：
   - 股票代码
   - 股票名称
   - 收盘价
   - 各指标数值
   - 综合评分

3. **排序**：
   - 默认按综合评分降序排列
   - 可点击表头进行排序

4. **交互功能**：
   - 点击行查看详细股票信息
   - 复制股票代码到剪贴板

### 8. 保存选股报告

点击「💾 保存报告」：

- 报告保存到`reports/`目录
- 文件名格式：`strategy_report_YYYYMMDD_HHMMSS.html`
- 报告包含完整的选股结果和参数配置

### 9. 回测分析

对选股结果进行回测：

#### 开始回测

1. 在选股结果页面点击「开始回测」
2. 配置回测参数：
   - 止盈比例
   - 止损比例
   - 回测天数
   - 初始资金
   - 手续费比例
3. 点击「执行回测」

#### 回测参数配置机制

系统使用统一的回测参数配置机制，确保策略控制器和参数优化器使用相同的参数定义和验证逻辑：

- **参数管理模块**：`strategy_controller/utils/backtest_params_manager.py` 集中管理所有回测参数
- **参数设置组件**：`strategy_controller/ui/backtest_params_component.py` 提供统一的参数设置界面
- **参数一致性**：两个模块使用相同的参数定义、验证规则和默认值
- **参数范围**：支持自定义参数范围，适应不同场景的需求

#### 最大持仓天数参数

**功能说明**：
- 最大持仓天数是一个卖出条件，与止盈止损条件并列，满足任一条件即触发卖出交易
- 买入当天不计算在持仓天数内，从下一个交易日开始计算

**设置范围**：
- 范围：1-365天
- 默认值：5天

**行业最佳实践**：
- **短线交易**：1-5天
- **中线交易**：5-30天
- **长线交易**：30-365天

**与止盈止损的关系**：
- 三个条件（止盈、止损、最大持仓天数）具有相同的优先级
- 满足任一条件即触发卖出交易
- 系统会根据当前市场情况自动选择最合适的卖出时机

#### 回测结果

回测完成后显示：

- 总收益率
- 年化收益率
- 最大回撤
- 夏普比率
- 胜率
- 交易次数
- 收益曲线图

#### 导出回测结果

点击「导出回测报告」：
- 生成详细的回测HTML报告
- 包含交易明细和图表分析

## 参数优化器使用

参数优化器用于寻找最优策略参数。

### 1. 启动参数优化器

#### 方法A：使用启动器（推荐）

```bash
python launcher.py
```

选择「2. 启动参数优化器」或「3. 同时启动两个应用」。

#### 方法B：直接启动

```bash
streamlit run ulti-para-seeker/app.py --server.port 8501
```

### 2. 算法配置

#### 支持的算法

**暴力搜索**：
- 遍历所有可能的参数组合
- 优点：保证找到全局最优解
- 缺点：计算量大，耗时较长
- 适用：参数空间较小的场景

**遗传算法**：
- 基于生物进化原理
- 通过选择、交叉、变异迭代优化
- 优点：快速收敛，适合大规模参数空间
- 缺点：可能陷入局部最优
- 适用：复杂参数优化

**粒子群算法**：
- 模拟鸟群觅食行为
- 通过个体和群体经验迭代优化
- 优点：收敛速度快，参数调节简单
- 缺点：对初始参数敏感
- 适用：中等规模参数优化

### 3. 参数设置

#### 止盈止损设置

```
止盈设置：
  最小值：3%
  最大值：15%
  步长：2%

止损设置：
  最小值：-1%
  最大值：-5%
  步长：-1%
```

#### 权重步长

设置权重调整的步长（5%-20%），步长越小，搜索越精细但耗时越长。

#### 测试模式

勾选后仅处理少量参数组合，用于快速验证配置。

#### 高级权重配置

启用后可以：
- 选择重点关注指标
- 设置重点指标权重因子（1.1-3.0倍）
- 重点指标在权重生成时获得更高权重

#### 其他参数

- **最大子权重组合数**：子权重的最大组合数量
- **初始资金**：回测的初始资金
- **回测天数**：回测的时间范围
- **回测终点日期**：回测的结束日期

### 4. 生成参数组合

点击「生成参数组合」：

1. 系统根据设置生成所有可能的参数组合
2. 显示总参数组合数和预计计算时间
3. 生成蓝图文件：`parameter_blueprint.json`

**蓝图信息**：
- 总组合数
- 生成时间
- 版本信息
- 算法类型

### 5. 蓝图管理

#### 查看蓝图

在「蓝图管理」标签页：
- 查看所有蓝图文件列表
- 查看每个蓝图的详细信息
- 查看蓝图状态统计（已完成/运行中/待处理）

#### 加载蓝图

加载已有的蓝图进行查看或继续优化：

1. 选择蓝图文件
2. 点击「加载蓝图」
3. 查看蓝图内容和状态

#### 重置蓝图

重置蓝图中所有组合的状态：

1. 选择要重置的蓝图
2. 点击「重置选中的蓝图」
3. 所有组合状态恢复为pending
4. 清除回测结果

**注意**：此操作不可恢复。

#### 删除蓝图

删除不需要的蓝图文件：

1. 选择要删除的蓝图
2. 点击「删除选中的蓝图文件」
3. 确认删除

**注意**：删除操作不可恢复。

#### 清空所有蓝图

一次性删除所有蓝图文件：

1. 点击「清空所有蓝图文件」
2. 再次点击确认删除

**注意**：此操作不可恢复，请谨慎使用。

### 6. 运行优化

点击「开始优化」：

1. 选择要优化的蓝图文件
2. 系统开始逐个测试参数组合
3. 实时显示优化进度
4. 完成后导出结果到Excel

#### 进度显示

- 进度条显示优化完成百分比
- 状态文本显示当前处理进度
- 支持`test_mode`模式快速测试

#### 测试模式

在测试模式下：
- 仅处理1个参数组合
- 用于快速验证配置
- 确认无误后再运行完整优化

### 7. 查看回测结果

点击「查看回测结果」：

#### 结果表格

显示所有完成的参数组合：

- 回测天数
- 回测日期
- 止盈止损比例
- 收益率指标
- 权重配置
- 子权重配置

#### 图表分析

- **收益率分布直方图**：展示所有组合的收益率分布
- **最优参数组合卡片**：显示收益率最高的参数配置

#### 下载结果

点击「📥 下载CSV」：
- 导出所有回测结果到CSV文件
- 文件名包含时间戳

### 8. 发送到策略控制器

将最优参数应用到策略控制器：

#### 发送最佳策略

1. 在「最优参数组合」卡片下方
2. 点击「发送最佳策略到控制器」
3. 最优参数会保存到策略控制器的配置文件
4. 在策略控制器中可以加载使用

#### 发送指定方案

1. 输入方案号（对应回测结果表格的序号）
2. 点击「发送指定方案到控制器」
3. 指定方案会保存到策略控制器
4. 在策略控制器中可以加载使用

## Token管理

### Token状态检查

系统会自动检查Token状态：

- ✅ **已配置**：Token已保存并可用
- ⚠️ **未配置**：需要配置Token才能使用

### Token操作

#### 保存Token

1. 输入Token（密码框，隐藏输入）
2. （可选）添加描述
3. 点击「保存Token」

#### 验证Token

点击「验证Token」：

- 检查Token格式（16-64字符）
- 测试Token连接

#### 更新Token

1. 在「Token管理」区域输入新Token
2. 点击「更新Token」

#### 删除Token

1. 在「Token管理」区域点击「删除Token」
2. 再次点击确认删除

**注意**：删除后需要重新配置Token。

### Token迁移

如果使用过旧版本的`token_config.py`：

1. 展开「从旧系统迁移」
2. 点击「迁移旧Token」
3. 系统自动检测并迁移

## 常见操作流程

### 流程1：快速选股

```
1. 启动应用
   ↓
2. 配置Token（首次）
   ↓
3. 选择策略（默认即可）
   ↓
4. 使用默认权重或选择已保存配置
   ↓
5. 设置筛选参数
   ↓
6. 点击「开始选股」
   ↓
7. 查看选股结果
```

### 流程2：参数优化

```
1. 启动参数优化器
   ↓
2. 配置算法和参数范围
   ↓
3. 点击「生成参数组合」
   ↓
4. 点击「开始优化」
   ↓
5. 等待优化完成
   ↓
6. 查看优化结果
   ↓

```

```
   ↓

   ↓
5. 使用参数优化器寻找最优参数
   ↓
6. 应用最优参数重新选股
   ↓
7. 再次回测验证
```

## 最佳实践

### 1. 选股参数设置

**保守策略**：
- 跳过ST股：勾选
- 最大结果数：20-30
- 权重偏重风险控制：`risk_reward`权重提高到25-30

**激进策略**：
- 测试模式：不勾选
- 最大结果数：50-100
- 权重偏重趋势和成交量：`trend`和`volume`权重提高

### 2. 参数优化建议

**首次优化**：
- 使用测试模式验证配置
- 选择较小的参数范围
- 使用暴力搜索算法

**深度优化**：
- 扩大参数范围
- 使用遗传算法或粒子群算法
- 增加最大子权重组合数

### 3. 权重配置技巧

**趋势市场**：
- 提高`trend`权重
- 降低`kdj_j`权重

**震荡市场**：
- 提高`kdj_j`权重
- 增加`deepv`权重

**基本面导向**：
- 提高`fundamental`权重
- 增加`position`权重

### 4. 回测建议

- 回测天数建议：30-180天
- 初始资金：10万-50万
- 止盈比例：5%-15%
- 止损比例：-3%到-8%
- 手续费比例：0.0003（万三）

### 5. 性能优化

**加快选股速度**：
- 增加并发数（6-12）
- 调整批量大小（500-2000）
- 使用测试模式快速验证

**减少内存占用**：
- 减少批量大小
- 清理旧缓存文件
- 关闭不需要的应用

## 故障排除

### 问题：选股结果为空

**可能原因**：
- 筛选条件过于严格
- 当前市场不满足策略条件
- Token权限不足

**解决方案**：
- 放宽筛选条件
- 检查市场状态
- 验证Token权限

### 问题：回测失败

**可能原因**：
- 选股结果为空
- 回测参数设置不当
- 数据获取失败

**解决方案**：
- 确保有选股结果
- 检查回测参数范围
- 查看错误日志

### 问题：优化速度太慢

**可能原因**：
- 参数空间过大
- 选择了暴力搜索算法
- 测试模式未开启

**解决方案**：
- 减小参数范围
- 使用遗传算法或粒子群算法
- 开启测试模式
