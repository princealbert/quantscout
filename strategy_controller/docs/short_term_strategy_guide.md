# 📈 QuantScout短期策略优化使用指南

## 🎯 功能概述

本系统基于现有QuantScout选股策略框架，增加了短期策略优化和回测功能，主要特点：

### ✨ 新增功能
- **短期策略优化**: 针对不同市场状况的优化配置
- **一键回测**: 直接发送选股结果到东财掘金回测系统
- **参数调优**: 自动化的短期策略参数优化
- **实时监控**: 策略表现实时跟踪和分析

### 🏗️ 技术架构
- **前端**: Streamlit Web界面
- **后端**: Python + 东财掘金API
- **数据**: 复用现有data、indicators、scoring模块
- **集成**: 与现有main.py回测系统无缝对接

## 🚀 快速开始

### 1. 启动前端界面
```bash
cd strategy_controller
streamlit run main.py
```

### 2. 配置短期策略
1. 在左侧面板选择"QuantScout综合策略"
2. 点击"短期策略优化"面板
3. 选择适合的优化策略：
   - **激进短期策略**: 高风险高回报
   - **稳健短期策略**: 平衡风险收益
   - **动量短期策略**: 捕捉短期动量

### 3. 执行选股
1. 配置权重参数
2. 设置筛选条件
3. 点击"开始选股"按钮
4. 等待选股完成

### 4. 发送到回测
1. 选股完成后，页面底部显示回测面板
2. 配置回测参数：
   - 回测天数（30/60/90/180天）
   - 初始资金
   - 回测股票数量
3. 点击"发送到回测"按钮
4. 生成回测脚本和配置文件

### 5. 运行回测
1. 打开东财掘金客户端
2. 加载生成的Python脚本
3. 配置API Token
4. 执行回测脚本
5. 查看回测报告

## 🔧 核心模块说明

### 1. 短期策略优化器 (`optimization/short_term_optimizer.py`)

**功能**: 提供针对短期交易的优化参数配置

**主要配置**:
- **激进策略**: 强调趋势和成交量，适合牛市
- **稳健策略**: 平衡各指标权重，适合大多数市场
- **动量策略**: 专注短期动量，适合震荡市

**使用方法**:
```python
from strategy_controller.optimization.short_term_optimizer import ShortTermOptimizer

optimizer = ShortTermOptimizer()
config = optimizer.get_optimization_config("aggressive_short_term")
```

### 2. 策略API接口 (`api/strategy_api.py`)

**功能**: 前端与回测系统之间的参数传递桥梁

**核心方法**:
- `create_backtest_package()`: 创建回测包
- `save_strategy_config()`: 保存策略配置
- `generate_backtest_script()`: 生成回测脚本

**数据流**:
```
前端界面 → API接口 → 配置文件 → 回测脚本 → 东财掘金
```

### 3. 回测组件 (`ui/backtest_component.py`)

**功能**: 前端回测功能集成

**特性**:
- 可视化回测参数配置
- 进度显示和状态监控
- 一键生成回测脚本
- 文件操作辅助功能

### 4. 优化组件 (`ui/optimization_component.py`)

**功能**: 短期策略优化界面

**特性**:
- 策略类型选择
- 参数可视化调整
- 实时配置预览
- 风险等级评估

## 📊 优化策略详解

### 激进短期策略
- **目标**: 高风险高回报
- **特点**: 高趋势权重(35%)，低KDJ权重(15%)
- **适用**: 牛市环境，经验丰富的交易者
- **风险等级**: 高

### 稳健短期策略
- **目标**: 平衡风险收益
- **特点**: 均衡各指标权重，注重风险控制
- **适用**: 大多数市场环境
- **风险等级**: 中

### 动量短期策略
- **目标**: 捕捉短期动量
- **特点**: 最高趋势权重(40%)，忽略基本面
- **适用**: 震荡市，短线交易
- **风险等级**: 高

## 🔍 使用技巧

### 1. 参数优化建议
- **牛市环境**: 使用激进策略，提高趋势权重
- **熊市环境**: 使用稳健策略，降低仓位
- **震荡环境**: 使用动量策略，控制仓位规模

### 2. 回测优化
- **短期验证**: 使用30天回测快速验证策略
- **中期评估**: 使用90天回测全面评估
- **长期测试**: 使用180天回测深度测试

### 3. 风险控制
- 设置合理的止损止盈比例
- 控制单只股票的仓位
- 定期评估策略表现
