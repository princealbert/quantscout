# 🚀 QuantScout量化选股系统：小白上手指南

## 一、系统简介

### 什么是QuantScout？

QuantScout是一个**基于东财掘金量化平台**的智能选股系统，核心特点：

- 🎯 **多维选股策略**：集成KDJ指标、知行趋势线、深V信号等多种技术指标
- ⚙️ **权重可调**：7大核心指标权重自由调整，重点指标可以放大
- 🔄 **参数优化**：3种优化算法（暴力/遗传/粒子群）自动寻找最优参数
- 📊 **可视化回测**：完整的策略回测功能，生成详细报告

> ⚠️ **重要声明**：本系统是一个**练习程序**，所有指标和因子组合仅为**散户自由搭配水平**，不具备专业投产策略的水准。核心目的是学习量化策略开发流程，而非提供可直接用于实盘交易的策略。

### 适合谁用？

- ✅ **量化小白**：想入门量化但不知道从哪开始
- ✅ **股票爱好者**：想用数据和算法辅助选股决策
- ✅ **编程初学者**：想学习量化策略开发的完整流程
- ✅ **专业投资者**：需要一个可定制的选股工具（注意：需自行优化策略）

## 二、准备工作

### 第一步：安装东财掘金量化终端

这是使用本系统的**前提条件**，一定要先做！

1. 访问东财掘金官网：https://gmsdk.gtja.com/
2. 注册账号并登录
3. 下载并安装「东财掘金量化终端」
4. 启动终端（后续使用时必须保持运行）

> ⚠️ **重要提示**：东财掘金量化终端必须始终运行，否则系统无法获取股票数据。

### 第二步：安装Python环境

1. 下载Python 3.8+：https://www.python.org/downloads/
2. 安装时勾选「Add Python to PATH」
3. 验证安装：打开命令提示符，输入 `python --version`

### 第三步：获取API Token

1. 打开东财掘金量化终端
2. 点击顶部菜单栏「系统」→「密钥管理」
3. 点击「生成Token」按钮
4. **复制生成的Token**（后面会用到）

## 三、启动系统

### 方式一：使用 Streamlit 多页面架构（推荐）

```bash

cd QuantScout

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动应用
```

启动后会打开浏览器，通过左侧导航栏切换"策略控制器"和"参数优化器"页面。

选择「3」同时启动两个应用，浏览器会自动打开。

>

## 五、策略控制器：选股流程详解

### 界面概览

页面分为两部分：

- **左侧边栏**：配置区域（策略选择、权重调整、筛选参数）
- **右侧主区域**：结果展示区域（选股结果、回测报告）

### 完整选股流程

#### 第一步：选择策略

在「选择选股策略」下拉框中选择：

- **多维综合策略**（推荐）：集成KDJ+知行趋势+深V信号的综合策略

#### 第二步：调整权重（核心操作）

权重决定了各指标在选股评分中的重要程度，总和为100分。

**7个核心指标说明：**

| 指标 | 说明 | 推荐场景 |
|------|------|----------|
| **KDJ J值** | 随机指标，判断超买超卖 | 短线操作、抓反弹 |
| **趋势** | 知行趋势线，判断上涨下跌 | 趋势交易、波段操作 |
| **深V信号** | 大幅下跌后的反弹信号 | 抄底、反转机会 |
| **成交量** | 成交量变化分析 | 判断资金动向 |
| **基本面** | PE等基本面指标 | 价值投资、长线持有 |
| **位置** | 股价在均线中的位置 | 判断相对高低位 |
| **盈亏比** | 风险收益比评估 | 风险控制 |

**权重调整技巧：**

- 📈 想抓超跌反弹：增大「KDJ J值」和「深V信号」权重
- 📊 做趋势交易：增大「趋势」权重
- 🛡️ 注重风险控制：增大「盈亏比」权重
- 💰 价值投资：增大「基本面」权重

> 💡 **小提示**：点击「🔧 高级配置 - 子指标权重」可以进一步细分每个指标的评分规则

#### 第三步：设置筛选参数

- **最大显示数量**：选出多少只股票（建议30-50只）
- **跳过ST股票**：建议开启，避开风险股票
- **测试模式**：只处理100只股票，快速验证配置是否正确

#### 第四步：执行选股

点击「🚀 开始选股」按钮，系统开始工作：

1. 页面显示进度条和状态消息
2. 启动项目的终端窗口显示详细日志
3. 等待选股完成（时间取决于股票池大小）
4. 完成后自动跳转到结果页

#### 第五步：查看结果

选股完成后，你会看到：

- 📋 **选股结果表格**：股票代码、名称、评分、买入信号等
- 📈 **回测按钮**：对选股结果进行回测分析
- 💾 **保存报告**：导出选股结果到Excel

## 六、参数优化器：寻找最优参数

如果你想进一步优化策略参数，可以使用参数优化器：

### 优化流程

1. 进入「参数优化器」页面
2. 配置优化算法（暴力搜索/遗传算法/粒子群算法）
3. 设置参数范围（止盈/止损/权重步长）
4. 点击「生成参数组合」创建蓝图
5. 点击「开始优化」执行搜索
6. 查看优化结果，找到最优参数组合
7. 点击「发送到策略控制器」应用最优参数

> 💡 **建议**：初学者可以先用暴力搜索，虽然慢但结果最可靠；熟练后可以尝试遗传算法和粒子群算法。

## 七、常见问题解答

### Q1：启动失败怎么办？

**A**：检查以下几点：
- 东财掘金量化终端是否正在运行
- Python版本是否为3.8+
- 依赖是否安装完整：`pip install -r requirements.txt`
- 端口是否被占用（默认8501、8502）

### Q2：Token验证失败？

**A**：
- 确保Token是从东财掘金终端正确复制的
- 检查Token格式是否正确（16-64个字符）
- 尝试重新生成Token
- 确保终端已登录

### Q3：选股时页面卡住？

**A**：这是正常现象！选股需要处理大量股票数据，页面会暂时卡住。你可以：
- 查看终端窗口的实时日志了解进度
- 启用测试模式快速验证配置
- 减小股票池范围（如选择沪深300而非全量A股）

### Q4：选股结果为空？

**A**：可能是参数设置太严格，建议：
- 增大「最大显示数量」
- 调整权重配置，不要过于集中在某一个指标
- 关闭测试模式，使用全量股票池

### Q5：如何保存我的配置？

**A**：
- 权重配置可以在左侧边栏保存、加载、复制
- 点击「💾 保存」按钮保存当前配置
- 配置会自动保存到本地文件
